## AGGREGATE FUNCTIONS

- `COUNT` - подсчитывает количество строк

```sql
SELECT
    COUNT(*)
FROM
    orders
```

- выведет количество заказов

- `SUM` - возвращает сумму все строк по колонке

```sql
SELECT
    SUM(sales)
FROM
    orders
```

- вернёт количество всех продаж в сумме

- `AVG` - возвращает среднее значение между всеми строками по колонке

```sql
SELECT
    AVG(sales)
FROM
    orders
```

- вернёт среднее количество продаж

- `MAX` - возвращает максимальное значение между строками по колонке

```sql
SELECT
    MAX(sales)
FROM
    orders
```

- вернёт максимальное количество продаж

- `MIN` - возвращает минимальное значение между строками по колонке

```sql
SELECT
    MIN(sales)
FROM
    orders
```

- вернёт минимальное количество продаж
